# Pretrained Model Path Map

This directory intentionally does not store model weights.

Use external checkpoints:

- FLUX generation backbone: `/data/zhengziyu/models/FLUX.1-dev`
- SigLIP image encoder: `/data/zhengziyu/Calligrapher/pretrained/siglip-so400m-patch14-384`
- Calligrapher adapter: `/data/zhengziyu/Calligrapher/pretrained/Calligrapher/calligrapher.bin`
- Optional FLUX-Fill inpainting backbone: `/data/zhengziyu/Calligrapher/pretrained/FLUX.1-Fill-dev`
- Shared local model pool, if needed: `/data/pretrained_models`
